# Signal Generator
MATLAB program to generate a custom signal and do multiple operations on it.

## Program Functionality

### Signal Types
1. DC Signal
2. Ramp Signal
3. General Order Polynomial
4. Exponential Signal
5. Sinusoidal Signal

### Operations
1. Amplitude Scaling
2. Time Shift
3. Time Reversal
4. Signal Compression
5. Signal Expansion
